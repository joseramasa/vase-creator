# Vase Creator

Parametric vase designer with real-time 3D preview. Built with React, Three.js, and TypeScript.

## Features

- **Parametric profile** — editable control points with Catmull-Rom interpolation
- **Surface patterns** — twist (spiral), vertical ribs, horizontal rings
- **Material presets** — clay, terracotta, glazed ceramic, porcelain, glass, bronze
- **Export** — STL for 3D printing, PNG screenshot
- **Bilingual** — English / Spanish

## Quick start

```bash
npm install
npm run dev
```

## Stack

React 19 · Three.js (react-three-fiber + drei) · Vite · TypeScript
